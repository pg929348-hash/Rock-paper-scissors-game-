# Rock Paper Scissors Game 🎮

A simple Python game where you can play Rock Paper Scissors against the computer. It also keeps track of the score!

## Features ✨
- **Random Computer Choice**: Computer picks `rock`, `paper`, or `scissors` using `random.choice()`
- **Score Tracking**: Tracks your wins, computer wins, and ties separately
- **Input Validation**: Handles invalid inputs and empty Enter presses
- **Exit Anytime**: Type `exit` to stop the game and see final score

1. Make sure Python 3 is installed on your PC/Laptop
